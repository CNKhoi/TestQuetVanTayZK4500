ZK4500 USB Direct Connection

Kết nối trực tiếp với ZK4500 qua USB HID Interface

Thông tin kết nối USB:

ZK4500 hỗ trợ 2 chế độ USB: HID Mode (Human Interface Device) và CDC Mode (Serial).

Ứng dụng này sử dụng WebUSB API để kết nối trực tiếp qua USB.
